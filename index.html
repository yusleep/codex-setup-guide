Setup Guide

Codex CLI 配置指南

三步完成安装与配置，即刻开始使用

1
注册获取令牌 →

注册账号后，在控制面板中获取你的 API 令牌。
该令牌将用于后续的身份验证配置。

2
安装 Codex CLI →

通过 npm 全局安装官方 Codex CLI 工具。

$ npm i -g @openai/codex
3
修改配置文件 →

分别编辑 ~/.codex/config.toml 和 ~/.codex/auth.json 两个文件。

config.toml 
model_provider = "bear2api"
model = "gpt-5.3-codex"  # 或者是 gpt-5.2
model_reasoning_effort = "xhigh"

[model_providers.bear2api]
name = "bear2api"
base_url = "https://bear2.xyz/v1"
wire_api = "responses"
auth.json 
{
  "OPENAI_API_KEY": "sk-dummy"
}

配置完成后，运行 codex 即可开始使用
